Глобальная цель: сконфигурировать приложение с различными правами доступа для разных страниц.
К странице home.html должны иметь доступ все аутентифицированные пользователи.
К странице home.html только пользователи с ролью 'ADMIN'
К странице /login - любые пользователи

Перед тем, как приступить к заданию, прогнать spring_security_db.sql для создания и заполнения базы.
В базе два пользователя:
    Admin с паролем password и ролью ADMIN
    User с паролем password и ролью USER

Ссылка на документацию:
https://docs.spring.io/spring-security/site/docs/current/reference/htmlsingle/#abstractsecuritywebapplicationinitializer

1. Добавить реализацию метода loadUserByUsername в UserServiceImpl
2. Сконфигурировать доступ пользователей в SecurityConfig
3. Создать собственную страницу логина взамен сгенерированной автоматически и зарегистрировать её
4. Реализовать возможность выхода из системы (logout)
